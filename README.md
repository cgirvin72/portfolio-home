# Analytics & BI Portfolio — Landing Page

The homepage for my analytics portfolio: short pitches, preview images, and links/downloads for each project.

**Live demo:** [portfolio-home-delta.vercel.app](https://portfolio-home-delta.vercel.app/)

## What's in this repo

- `index.html` — the portfolio landing page (static, no build step)
- `render.js` — project content and card rendering
- `data_images.js` — embedded preview images (base64, no external image hosting needed)
- `KPI_Definition_Governance_Workbook.xlsx` — KPI Definition & Governance Workbook (Consulting Operations KPI Program)
- `Consulting_Operations_Monthly_Review.pptx` — companion monthly operations review deck
- `KPI_Program_Methodology_Note.docx` — methodology note for the KPI program project

## Projects featured

1. **Vendor Consolidation Analysis** — links out to the separate `Analytics-Portfolio` repo's live interactive dashboard, plus the Excel model and supporting docs
2. **Consulting Operations KPI Program** — KPI governance workbook, monthly review deck, and methodology note, all downloadable directly from this page

## Data

All figures across every project are synthetic and representative. No actual employer, client, or proprietary data is used anywhere on this site.

## Tech

Single static HTML page with two small JS files. No build step, no dependencies — open `index.html` directly or deploy as a static site (Vercel, Netlify, GitHub Pages all work with zero configuration).
